## 🍽️ Restaurant Management System

A console-based Restaurant Management System developed in C that helps manage restaurant operations through a simple command-line interface. 

## 📖 Overview

This project was created to simulate the core functionalities of a restaurant management system. It allows users to manage , process information, and perform various operations to restaurant data through a menu.

## ✨ Features
Add new restaurant records
View all restaurant records
Search restaurants by specific criteria
Edit existing restaurant information
Delete restaurant records
Sort restaurant data using different algorithms
Read data from CSV files
Save updated data to files
Menu-driven user interface
Input validation and error handling

## 🛠️ Built With
Language: C
Compiler: GCC
IDE: Dev-C++
Data Storage: CSV Files
